# Finance Dashboard UI

A React-based finance tracker built with **Vite**, **Tailwind CSS**, and **Zustand**.

## 🚀 Features
- **Summary Cards:** Real-time calculation of Income, Expenses, and Balance.
- **Visual Insights:** Interactive Line and Pie charts using Recharts.
- **RBAC (Role-Based Access Control):** - **Admin:** Can add and delete transactions.
  - **Viewer:** Read-only access to the dashboard.
- **Responsive Design:** Optimized for mobile, tablet, and desktop.

## 🛠️ Tech Stack
- **Frontend:** React 18+ (Vite)
- **Styling:** Tailwind CSS v4
- **State Management:** Zustand
- **Icons & Charts:** Lucide-React & Recharts

## 📦 Installation
1. `npm install`
2. `npm run dev`
