# Zorvyn – Finance Dashboard UI
A modern finance tracking dashboard built using **React (Vite)**, **Tailwind CSS v4**, and **Zustand**.

## 🚀 Features

* 📊 **Summary Cards:** Real-time calculation of Income, Expenses, and Balance
* 📈 **Charts:** Interactive Line & Pie charts using Recharts
* 🔐 **RBAC (Role-Based Access Control):**

  * **Admin:** Can add and delete transactions
  * **Viewer:** Read-only access
* 📱 **Responsive Design:** Works on mobile, tablet, and desktop

## 🛠️ Tech Stack
* **Frontend:** React 18+ (Vite)
* **Styling:** Tailwind CSS v4
* **State Management:** Zustand
* **Charts:** Recharts
* **Icons:** Lucide React

## 📦 Installation
```bash
npm install
npm run dev
